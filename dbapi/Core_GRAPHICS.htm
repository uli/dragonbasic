CORE :: GRAPHICS

Sets the current graphics mode. 
SUB GRAPHICS(
    mode,
    sprites
)
Parameters
mode - the mode to set (must be 0-5)
sprites - a boolean flag whether or not sprites should be enabled (0=no, non-0=yes)
Return
None.
Remarks
Sets the current graphics mode. 0-2 are tiled modes and 3-5 are bitmapped modes. This function will completely erase the VRAM before returning.
