SPRITES :: SPRITE

Get the address of a copied sprite. 
FUNCTION SPRITE(
    n
)
Parameters
n - the sprite (0-127)
Return
The address of a sprite's copied OAM data.
Remarks
All sprite (OAM) data is duplicated in RAM. You should use SPRITE to get the address of the copied data so you are manipulating data that will be upated with the next UPDATESPRITES. If you directly manipulate OAM data, UPDATESPRITES will erase all your changes.
