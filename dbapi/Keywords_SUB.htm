SUB name &optional ([arg1,arg2,...])

Description

Declares a new sub.

Results

Creates a new sub in ROM with name. Optionally, parameters can be passed to the sub to make use of.

Example

; create a sub to sum a list of numbers
sub sum_array(list,count)
    total = 0 ; <-- total is global
    for x = 0 to count - 1
        total = total + list[x]
    next 
    return ; optionnal
end sub
