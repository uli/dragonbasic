CORE Subroutines & Functions

COPY - Copies bytes of data from one address to another.
ERASE - Erases all memory at a given address, filling it with 0.
GRAPHICS - Sets the current graphics mode.
LOG - Writes a string to the console of VBA.
PEEK - Loads a value from RAM.
POKE - Writes a value to RAM.
SCANLINE - Returns the current scanline begin drawn.
VBLANK - Waits for the next vertical blank to begin.
