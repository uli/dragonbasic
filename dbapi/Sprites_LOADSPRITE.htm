SPRITES :: LOADSPRITE

Copies bitmap data from ROM to VRAM. 
SUB LOADSPRITE(
    char,
    label,
    blocks
)
Parameters
char - the sprite character block (0-1023)
label - the address of the imported data
blocks - the number of blocks the image occipies
Return
None.
Remarks
Copies data imported with the #BITMAP directive to VRAM (sprite character data).
