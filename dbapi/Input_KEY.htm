INPUT :: KEY

Returns a masked copy of the current input register. 
FUNCTION KEY(
    mask
)
Parameters
mask - a bit-mask of all buttons to test for, OR'ed together with NOR
Return
Reads REG_P1 from GBA hardware register, masks and returns.
Remarks
Use this function to check for 1 or more specific buttons being pressed. A non-zero value indicates the button(s) is/are pressed and 0 is released.
