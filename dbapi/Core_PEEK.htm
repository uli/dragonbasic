CORE :: PEEK

Loads a value from RAM. 
FUNCTION PEEK(
    address
)
Parameters
address - a 32-bit, unsigned address to load from
Return
A 16-bit, halfword value.
Remarks
Loads a halfword (16-bit) value from GBA RAM. Typically used with VRAM for pixel data or GBA register data.
