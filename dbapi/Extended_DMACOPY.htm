EXTENDED :: DMACOPY

Copies words (32-bit) of data from one address to another. 
SUB DMACOPY(
    dest,
    source,
    words
)
Parameters
dest - a 32-bit, unsigned destination address
source - a 32-bit, unsigned source address
words - the number of words (32-bit) to copy from source to dest
Return
None.
Remarks
Uses DMA 3 to copy words of data. To copy bytes, use COPY.
