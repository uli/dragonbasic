TILES :: ORDERTILES

Set the z-order of a background layer. 
SUB ORDERTILES(
    bg,
    priority
)
Parameters
bg - the background layer (0-3)
priority - the z-order priority (0-3)
Return
None.
Remarks
0 is the highest priority (always drawn on top) and 3 is the lowest priority. Sprites with the same priority as a background layer are drawn over the background.
