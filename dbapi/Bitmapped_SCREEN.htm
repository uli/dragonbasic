BITMAPPED :: SCREEN

Returns the currently hidden buffer in a double-buffer mode. 
FUNCTION SCREEN
Parameters
None.
Return
The 32-bit, unsigned address to the currently hidden buffer.
Remarks
In modes 4 and 5, this function will either return &6000000 or &600A0000. In mode 3, this function always returns &6000000.
