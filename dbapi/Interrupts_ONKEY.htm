INTERRUPTS :: ONKEY

Creates or halts a keypad interrupt control. 
SUB ONKEY(
    i,
    mask,
    flag
)
Parameters
i - the interrupt to be called or 0 to disable
mask - a bit-mask of all buttons to test for, OR'ed together with NOR
flag - false if any button will fire the interrupt, or true if all buttons must be pressed to fire it
Return
None.
Remarks
Use this function to setup a keypad interrupt control function. This function is good for a pause routine or when the control of your game is simple.
