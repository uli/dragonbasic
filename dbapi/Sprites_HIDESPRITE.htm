SPRITES :: HIDESPRITE

Hide a sprite so that it isn't displayed. 
SUB HIDESPRITE(
    n
)
Parameters
n - the sprite index (0-127)
Return
None.
Remarks
Moves the sprite offscreen (240,160) so that it isn't visible.
