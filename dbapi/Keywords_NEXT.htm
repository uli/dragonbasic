NEXT

Description

End a FOR loop.

Results

Compiles the branch back to a FOR statement after incrementing var by the STEP value of the FOR loop.

Example

; loop through all the indices of a loop, skipping every other one
for x = 0 to 99 step 2
    ; quit the loop if x > 50
    if x > 50 then  x = 99
next
