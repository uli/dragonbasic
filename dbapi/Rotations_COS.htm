ROTATIONS :: COS#

Look up the cosine of an angle. 
FUNCTION COS#(
    degrees
)
Parameters
degrees - an angle between 0-359
Return
The 24:8 fixed-point representation of the cosine of an angle.
Remarks
The angle must be in degrees and between 0-359.
