STRINGS :: COMPARE

Compare two strings for equality. 
FUNCTION COMPARE(
    string1$,
    string2$
)
Parameters
string1$ - the string being compared
string2$ - the string that it is compared to
Return
Returns -1 if string1$ is less than string2$, 0 if they are the same and +1 if string1$ is greater than string2$.
Remarks
Only the bytes of string1$ and string2$ are tested, not the length. The compare is case sensitive.
