CLOCKTIMER

Checks how many timer the user-defined timer has fired.

Arguments

No arguments

Results

Returns the number of times the timer has fired since the last RESETTIMER.

Example

; make a timer that fires 10 times a second
maketimer 10
starttimer

; do nothing until 5 seconds has elaped
while clocktimer < 50
loop
