BITMAPPED :: BLIT

Copies a bitmap image onto the screen with a mask. 
SUB BLIT(
    label,
    x,
    y,
    width,
    height
)
Parameters
label - the address of the imported image
x - the x coordinate of the upper-left destination pixel
y - the y coordinate of the upper-left destination pixel
width - the width of the image to BLIT in pixels
height - the height of the image to BLIT in pixels
Return
None.
Remarks
Pastes the image at label (imported with #BITMAP) onto the screen. In modes 3 and 5, RGB(0,0,0) is not pasted, but is used as a mask. In mode 4, palette index 0 is the mask.
