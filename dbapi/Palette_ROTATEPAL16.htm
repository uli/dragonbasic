PALETTE :: ROTATEPAL16

Rotates all the colors in a palette. 
SUB ROTATEPAL16(
    palette,
    index
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
index - the 16-color palette index to rotate (0-15)
Return
None.
Remarks
Rotates the upper 15-color entries in the 16-color palette defined by palette and index, leaving the transparency color alone. Excellent for palette animations!
