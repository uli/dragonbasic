TILES :: MAPIMAGE

Copy a map from ROM to a screen block. 
SUB MAPIMAGE(
    address,
    base,
    width,
    height
)
Parameters
address - an unsigned, 32-bit address returned by TILE
base - the index of the first tile to write
width - the number of tiles horizontally to map
height - the number of tiles vertically to map
Return
None.
Remarks
Starting at address, and bounded by width and height, base index is written, then incremented by 1 for each successful write. Mapping is done left to right, top to bottom. Only the tile is set! No palette or flip information is written.
