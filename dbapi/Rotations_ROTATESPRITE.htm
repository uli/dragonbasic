ROTATIONS :: ROTATESPRITE

Enables the rotation of a sprite. 
SUB ROTATESPRITE(
    n,
    matrix
)
Parameters
n - the sprite index (0-127)
matrix - a rotation matrix (0-3)
Return
None.
Remarks
Passing a negative rotation matrix will turn off rotations for the sprite. Once the sprite is attached to a rotation matrix, to rotation the sprite you only need to recreate the rotation matrix with MAKEROTATION.
