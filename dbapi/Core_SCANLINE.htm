CORE :: SCANLINE

Returns the current scanline begin drawn. 
FUNCTION SCANLINE
Parameters
None.
Return
An unsigned value from 0-227.
Remarks
Scanlines 0-159 are visible on the screen. Vertical blanks occur at scanline 160.
