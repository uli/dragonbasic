CORE :: VBLANK

Waits for the next vertical blank to begin. 
SUB VBLANK
Parameters
None.
Return
None.
Remarks
Waits for the current scanline to reach 160 (bottom of the screen) before returning. The GBA refresh rate is 60 Hz (60 vblanks occur per second).
