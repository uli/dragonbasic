PALETTE :: GETPALENTRY

Gets the 15-bit, RGB color value of a palette entry. 
FUNCTION GETPALENTRY(
    palette,
    index,
    entry
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
index - the 16-color palette index to rotate (0-15)
entry - the actual color entry to set (0-15)
Return
A 15-bit, RGB color value.
Remarks
Gets the color value of an entry in the 16-color palette defined by palette and index.
