STRINGS :: LEN

Calculates the length of a string. 
FUNCTION LEN(
    string$
)
Parameters
string$ - the string to get the length of
Return
Returns the length of the string (0-255 characters).
Remarks
The length is always stored in the first byte of a string. Immediately following the length byte are the bytes of the string.
