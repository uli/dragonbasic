TILES :: FLIPTILE

Flips or mirrors a tile. 
SUB FLIPTILE(
    address,
    x,
    y
)
Parameters
address - an unsigned, 32-bit address returned by TILE
x - flag to mirror the tile about the y-axis (horizontally).
y - flag to flip the tile about the x-axis (vertically).
Return
None.
Remarks
x and y should be 0 to disable the appropriate flip/mirror bits and non-0 to enable them. If multiple tiles use the same data, flipping one will not flip the others.
