READ var1 &optional ,var2 &optional ...

Description

Reads data from the current data pointer.

Results

Once the data pointer has been set with RESTORE, data may be READ from the ROM into variables. Each READ increments the data pointer by 4 bytes. Note:All data read is 32-bit (even if stored as 16-bit).

Example

; create some data
my_data:
    data 1,2,3,4,5


; create an array
dim x(5)

start:
    ; restore and read the data
    restore my_data
    for k = 0 to 5
        read x[k]
    next
