EXTENDED :: FIX#

Converts an integer to a fixed-point value. 
FUNCTION FIX#(
    n
)
Parameters
n - an integer
Return
The fixed-point equivelant of n.
Remarks
Converts n to a fixed-point value by bit-shifting left the whole value.
