EXTENDED :: ROUND

Rounds a fixed-point value up or down and returns an integer. 
FUNCTION ROUND(
    f#
)
Parameters
f# - a fixed-point value
Return
The whole portion of the rounded value of f#.
Remarks
First rounds f# up or down to the nearest whole number. Then returns the whole value as an integer.
