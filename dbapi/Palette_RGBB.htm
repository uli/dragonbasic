PALETTE :: RGBB

Gets the blue component from a 15-bit, RGB color value. 
FUNCTION RGBB(
    color
)
Parameters
color - a 15-bit, RGB color value
Return
The 5-bit, blue component of color.
Remarks
The component is masked and shifted down to the correct level for a value in the range of 0-31.
