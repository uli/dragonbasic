INTERRUPTS :: ONVCOUNT

Creates or halts a vertical counter interrupt control. 
SUB ONVCOUNT(
    i,
    n
)
Parameters
i - the interrupt to be called or 0 to disable
n - the scanline on which to fire the interrupt
Return
None.
Remarks
This interrupt will fire each time the current scanline reaches n. Can be used as a second VBLANK interrupt or for effects.
