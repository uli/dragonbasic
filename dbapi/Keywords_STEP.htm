STEP [expr]

Description

Sets the step value for a FOR loop.

Results

Overrides the default step value in a FOR loop with [expr]. Note: if DOWNTO is used, make sure that [expr] is a negative value.

Example

; sum the odd numbers from 1 to 100
sum = 0
for x = 1 to 100 step 2
    sum = sum + x
next
