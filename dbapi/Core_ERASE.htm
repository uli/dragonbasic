CORE :: ERASE

Erases all memory at a given address, filling it with 0. 
SUB ERASE(
    address,
    bytes
)
Parameters
address - a 32-bit, unsigned address
bytes - the number of bytes to erase
Return
None.
Remarks
Writes 0 to a number of bytes starting at address. If bytes is odd, at lease bytes+1 bytes will be erased.
