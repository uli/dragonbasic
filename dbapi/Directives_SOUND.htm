DIRECTIVES :: #SOUND

Import an 8- or 16-bit WAV file. 
#SOUND "file"
Parameters
None.
Return
None.
Remarks
The WAV file will be converted from 16-bit or stereo to 8-bit, mono sample to help save space, and can be played back with PLAYMUSIC or PLAYSOUND.
