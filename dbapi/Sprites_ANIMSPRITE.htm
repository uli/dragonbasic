SPRITES :: ANIMSPRITE

Animates a sprite. 
SUB ANIMSPRITE(
    n,
    first,
    last,
    blocks
)
Parameters
n - the sprite index (0-127)
first - the first sprite character in the animation (0-1023)
last - the last sprite character in the animation (0-1023)
blocks - the number of blocks between animations
Return
None.
Remarks
If the current sprite frame is outside of the animation it will be reset to first. Otherwise the frame will increment by blocks until > last, at which point it will reset back to first.
