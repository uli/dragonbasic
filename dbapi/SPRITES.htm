SPRITES Subroutines & Functions

ANIMSPRITE - Animates a sprite.
BUMPSPRITES - Checks to see if two sprites overlap.
COLORSPRITE - Sets the palette that a sprite should use.
FLIPSPRITE - Flips or mirror a sprite when drawn.
HIDESPRITE - Hide a sprite so that it isn't displayed.
LOADSPRITE - Copies bitmap data from ROM to VRAM.
MAKESPRITE - Setup a new sprite object.
MOVESPRITE - Move a sprite's position relatively.
ORDERSPRITE - Set the z-order of a sprite.
POSITIONSPRITE - Set the absolute position of a sprite
SIZESPRITE - Sets the hardware drawing size of a sprite.
SPRITE - Get the address of a copied sprite.
SPRITEFRAME - Get the current sprite character being drawn.
SPRITEHIDDEN - Check to see if a spr ite is onscreen.
SPRITEX - Get the x coordinate of a sprite's position.
SPRITEY - Get the y coordinate of a sprite's position.
UDPATESPRITES - Updates all OAM data.
