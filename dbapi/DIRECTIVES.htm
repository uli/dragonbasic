DIRECTIVES Subroutines & Functions

#IMPORT - Import a binary file.
#SOUND - Import an 8- or 16-bit WAV file.
#CONSTANT - Creates a constant macro for the compiler.
#REQUIRES - Imports a TIN file from the /optional folder.
#INCLUDE - Include another Dragon BASIC source file.
#PALETTE - Import a palette from BMP, PCX or PAL formatted files.
#BITMAP - Import a BMP or PCX file.
#TITLE - Set the title of the compiled ROM image.
#MUSIC - Not implemented.
