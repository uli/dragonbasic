BITMAPPED :: WALLPAPER

Copies a fullscreen image from ROM to VRAM. 
SUB WALLPAPER(
    label
)
Parameters
label - the address to a 24-bit or 8-bit image imported with #BITMAP
Return
None.
Remarks
Uses DMA 3 to copy image data located at label to the current screen or back buffer. In mode 4, the image is 8-bit, while in modes 3 and 5 the image is 24-bit.
