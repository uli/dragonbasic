STRINGS :: APPEND$

Appends one string to another, returning a new string. 
FUNCTION APPEND$(
    string1$,
    string2$
)
Parameters
string1$ - the beginning portion of the new string
string2$ - the string to append to the beginning
Return
A new string which is string1$ + string2$.
Remarks
If the lengths of string1$ and string2$ combine to more than 255, then the result is undefined. Otherwise, the return value can be copied into a new string.
