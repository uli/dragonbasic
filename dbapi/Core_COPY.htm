CORE :: COPY

Copies bytes of data from one address to another. 
SUB COPY(
    address,
    source,
    bytes
)
Parameters
address - a 32-bit, unsigned destination address
source - a 32-bit, unsigned source address
bytes - the number of bytes to copy
Return
None.
Remarks
Copies bytes from source to address. If bytes is odd, at least bytes+1 bytes will be copied.
