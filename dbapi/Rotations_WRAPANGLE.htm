ROTATIONS :: WRAPANGLE

Clamps an angle to [0,359]. 
FUNCTION WRAPANGLE(
    degrees
)
Parameters
degrees - an angle between 0-359
Return
An angle between the range of 0 and 359.
Remarks
This function should only be used when modifying an existing angle that may end above 359 or below 0. All trigometric functions in Dragon BASIC takes angles between 0 and 360 degrees.
