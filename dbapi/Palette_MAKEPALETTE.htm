PALETTE :: MAKEPALETTE

Creates a web-safe, 216-color palette in RAM. 
SUB MAKEPALETTE(
    palette
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
Return
None.
Remarks
The palette created is web-safe and well known. Red, green and blue components are all used and incremented individually for a very smooth palette. The last 40 colors are available for you to use at will.
