SPRITES :: SPRITEY

Get the y coordinate of a sprite's position. 
FUNCTION SPRITEY(
    n
)
Parameters
n - the sprite index (0-127)
Return
The current (copied) y-coordinate of the sprite's position.
Remarks
If offscreen, this position will be negative (even if off to the bottom of the screen).
