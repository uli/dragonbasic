PALETTE :: LOADPAL256

Loads a 256-color palette into RAM. 
SUB LOADPAL256(
    palette,
    label
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
label - address of the palette imported with #PALETTE
Return
None.
Remarks
Copies 256, 15-bit, RGB colors from label into the 256-color palette defined by palette.
