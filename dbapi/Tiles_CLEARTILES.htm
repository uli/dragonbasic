TILES :: CLEARTILES

Erase an area of tiles. 
SUB CLEARTILES(
    adress,
    width,
    height
)
Parameters
adress -
width - the number of tiles horizontally to erase
height - the number of tiles vertically to erase
Return
None.
Remarks
Sets the tile being used by the map in the defined box to 0, as if set with SETTILE.
