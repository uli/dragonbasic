TILES :: CHARBLOCK

Gets the address of a character block. 
FUNCTION CHARBLOCK(
    n
)
Parameters
n - a character base block (0-3)
Return
The address of a character base block in VRAM.
Remarks
Character base blocks are used for tile data. They hold all tile graphics that screen blocks then use for mapping.
