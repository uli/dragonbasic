INPUT :: INPUT

Waits for a button to change state. 
SUB INPUT(
    mask
)
Parameters
mask - a bit-mask of all buttons to test for, OR'ed together with NOR
Return
None.
Remarks
Waits for any button in mask to change state (be pressed or be released). This is a good method of just waiting for a button press or release when you already know the state of the button.
